Beta Information
Hello (hopefully) voluntary beta tester. Welcome to Campus Finder, an application that helps students navigate their way through campus. This application is a year-long project for the Engineering, Design, and Development (EDD) class of eight students total!

Our intention is to have current students conduct tests beyond our internal ones. To return to this page later, simply click on the User icon in the top right and select "Beta Information." Beta Tester Tasks:
  1. Search for a teacher with the search bar and see if their room location is relatively accurate
  2. Search for a building at West and ensure that its location is accurate and that it works.
  3. Login with Google and input your classes.
  4. Check if the path outlined in the Google map is reasonable.

Thanks,
The Campus Finder Team: Jason A., Ian D., and Jay M.
