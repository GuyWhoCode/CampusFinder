# Campus Finder
Customizable Map locator specifically made for school campuses.

## Inspirations:
- [UCLA Map](http://www.map.ucla.edu/)
- [MapMe](https://mapme.com/solutions/virtual-campus-tour/)

## Dependencies
- Node JS
- MongoDB 
- dotenv 
- Express
- Graceful-fs
- Socket.io
- Firebase
- Bootstrap

## Dev Dependencies
- ESLint
- Nodemon
